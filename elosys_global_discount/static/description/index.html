Fonctionnalités Principales

  • A la création d’un nouveau document de vente, devis, bon de commande ou facture en brouillon, vous avez à disposition un champ nommé « Remise globale ».
  • Ce champ vous permet d’appliqué une remise sur le montant global de votre transaction.
    • Cette remise est répercutée sur l’ensembles des articles de votre transaction.

NOTE : ce champ est visible pour tous les utilisateurs mais ne peut être manipulé que par les utilisateurs aillant les droits nécessaires sur odoo (Voir Implémentation)


Implémentation

  • Le module est disponible en :
    • Anglais.
    • Français ;
  • Pour accéder à cette fonctionnalité il faut :
    • Passé en mode développeur
    • Dans le menu configuration du module vente, cochez la case « Remise ».
    • Dans le menu configuration des utilisateurs, cochez la case « Peut définir une remise globale »

    Voir l’onglet « SREENSHOTS »
  • Vous verrez, à ce moment-là, apparaitre sur vos formulaires de vente, le champs « Remise Globale ».
  • Ce champ permet d’introduire un pourcentage de remise à appliquer sur le montant global.
    • Le champ « Remise globale » est repris d’un formulaire à l’autre selon la progression native des formulaires de odoo.
      - Devis → Bon de commande → Facture brouillon → Facture validé
  • Le champ « Remise globale » est modifiable sur les devis et factures a l’état brouillon.
  • Le champ « Remise globale » est en lecture seule sur les bons de commandes et les factures validés
  • Le champ « Remise globale » est en lecture seule sur tous les formulaires pour les utilisateurs n’aillant pas le droit de le modifié.

Impressions & Rapports

  • La valeur de la remise globales est visible sur les documents de ventes si on les imprime :
    • Impression d’un devis ou d’un bon de commande
    • Impression d’une facture avant et après validation

    Voir l’onglet « SREENSHOTS »

Scénarios

  • Après avoir tapé une valeur de remise globale non nulle puis sauvegarder.
  • Celle-ci se répercutes sur tous les articles.
    • Le montant est recalculé en fonction des nouvelles valeurs.
  • Si vous modifié une des valeurs des remises partielles (propre à chaque article).
    • Le montant de la remise globale redevient nul.
    • Les remises appliquées sur autres lignes de commandes restent telles qu’elles sont.
    • Le montant est recalculé en fonction des nouvelles valeurs.
  • Si vous modifié à nouveau le montant de la remise Globale
    • Cette valeur est reprise sur Les toutes les remises partielles (pas articles).
    • Le montant est recalculé en fonction des nouvelles valeurs.

    Voir l’onglet « SREENSHOTS »

NOTE : Le montant de la TVA à la vente est recalculé à chaque modification, que ce soit sur les remises partielles ou sur la remise globale.
